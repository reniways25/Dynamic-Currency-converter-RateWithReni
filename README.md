# Dynamic-Currency-converter-RateWithReni

RateWithReni 💱
A lightweight, real-time currency converter built with HTML, CSS, and vanilla JavaScript.
Features

Convert between 10 major currencies (USD, EUR, GBP, JPY, AUD, CAD, CHF, CNY, INR, NGN)
Live rate simulation with automatic updates every 5 seconds
Swap currencies instantly with the ⇄ button
Formatted number input (e.g. 1,000,000) for easy readability
Clean dark UI with smooth animations

How to Use

Open index.html in any modern browser — no install or build step needed.
Select your from and to currencies using the dropdowns.
Type an amount in the Your payment field and the converted value updates instantly.
Hit ⇄ to swap the two currencies.
